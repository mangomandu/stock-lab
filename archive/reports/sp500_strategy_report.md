## TL;DR

> **S&P 500로 universe 확장 + 31 windows walk-forward에서 ML 알파 +18.63%p vs SPY (t=5.69, p<0.0001), 승률 28/31 (90%). 모든 검증 통과한 가장 robust한 결과. Event-driven 룰들은 대부분 효과 없거나 손해. 이미 ML이 자연스럽게 위기 대응 중.**

## 4. Event-driven 룰 백테스트 (검증 완료)

### 결과 (Sharpe 정렬, 31 windows)

| 설정 | CAGR | Sharpe | MDD | vs SPY | Win | t-stat |
|---|---|---|---|---|---|---|
| **★ Buy dip 5d-3% → Top-25** | +31.50% | **1.36** | -18.72% | **+19.26%p** | 29/31 | 5.72 |
| Buy dip 5d-5% → Top-30 | +31.43% | 1.36 | -18.75% | +19.19%p | 27/31 | 5.58 |
| **Baseline (룰 없음)** | +31.28% | 1.35 | -18.73% | +19.05%p | 27/31 | 5.74 |
| Vol filter (>2.5%/일) | +29.78% | 1.33 | -18.19% | +17.54%p | 28/31 | 5.80 |
| Drawdown halt (-20%) | +29.75% | 1.32 | -18.01% | +17.52%p | 28/31 | 5.84 |
| Stop-loss -20% | +29.44% | 1.31 | -18.56% | +17.20%p | 26/31 | 5.35 |
| Stop-loss -15% | +26.97% | 1.25 | -18.38% | +14.73%p | 24/31 | 5.04 |
| **Stop-loss -10%** | +23.36% | 1.16 | -18.02% | +11.12%p | 24/31 | 4.06 |

### 핵심 결론

1. **Baseline (룰 없음)이 거의 최고** — Sharpe 1.35, 알파 +19.05%p
2. **Buy-the-dip 5d-3% → Top-25** 미세 개선 — Sharpe 1.36 (+0.01), 승률 29/31 (90%)
3. **Stop-loss 명백히 손해** — -10% 사용 시 알파 8%p 떨어짐 (회복기 놓침)
4. **Vol filter / Drawdown halt** 모두 -2~3%p 알파 손실 (위기 회피하다 회복 놓침)

### 권장 운용 룰 (확정)

```
방법 A: Baseline 그대로 (단순)
- 격주 Top-20 동일가중 매매
- 추가 룰 없음
- Sharpe 1.35, 알파 +19.05%p

방법 B: Buy-the-dip 추가 (선택, 미세 개선)
- 평소: Top-20 동일가중 매매
- 5일 SPY -3% 이상 폭락 시 다음 매매에서 Top-25로 확장
- Sharpe 1.36 (+0.01), 알파 +19.26%p
```

→ 방법 A 권장 (단순함). Buy-the-dip은 운용 부담 ↑ 대비 개선 미미.

데이터 검증 결과 **Stop-loss / Volatility filter / Drawdown halt 모두 효과 없거나 손해**. 룰 추가 권장 X.

### 단순 운용 룰 (확정)

```
1. 매년 1월: 최근 5년 데이터로 ML 모델 학습
2. 매주 1회 이상: 점수 갱신 (자유)
3. 격주 (10거래일) 매매:
   - ML 점수 Top-20 추출
   - 동일가중 (1/20 = 5%씩)
   - 직전 portfolio와 차이만 매매
4. 추가 룰: 없음 (검증 결과)
```
